Nodenberg API Documentation

This site is the API documentation portal for the Nodenberg Excel/PDF generation server.

What this API provides​

  • Placeholder detection in Excel templates
  • Template metadata and sheet discovery
  • Excel generation from template + JSON data
  • PDF generation through LibreOffice
  • Legacy and section-table placeholder expansion
